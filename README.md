### 技术选型

#### 前端技术:
技术 | 名称
----|------
React | 前端框架  
Redux | 状态管理  
Redux-Sagas | Redux异步操作中间件
React-Router | 路由
Immutable | 不可变数据
antd | UI
moment | 日期转化
classnames | 样式控制

#### [后端](https://github.com/DigAg/digag-server)技术:
技术 | 名称 
----|------
Spring Framework | 容器
SpringBoot | MVC框架
Spring Security | 安全框架
Hibernate | ORM框架
Druid | 数据库连接池
Redis | 分布式缓存数据库
Log4J | 日志组件
Swagger2 | 接口测试框架
Docker | 应用容器
Maven | 项目构建管理
